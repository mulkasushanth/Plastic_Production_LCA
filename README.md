Plastic Production Life Cycle Assessment using openLCA

This project presents a simplified cradle-to-gate Life Cycle Assessment (LCA)
of 1 kg of plastic production developed using openLCA.

Functional Unit
---------------
1 kg Plastic Product

Processes Included
------------------
• Crude Oil Extraction
• Naptha Production
• Steam Generation
• Diesel Production
• Electricity Generation
• Ethylene Production
• Polymerization
• Plastic Production

Software
--------
• openLCA 2.6.2

Goal
----
To estimate the cradle-to-gate carbon dioxide emissions associated with plastic production.

Inventory
---------
Inputs
• Crude Oil
• Electricity
• Steam
• Diesel
• Water

Output
------
1 kg Plastic Product

Results
-------
Total CO₂ emissions:
≈4.18 kg CO₂ per kg of plastic produced

Largest contributors:
• Crude Oil Extraction
• Ethylene Production
• Naptha Production
• Steam Generation
